Singleton Pattern with Private Constructor

The BankServer constructor is private, preventing direct instantiation. All instances are created through the static getInstance() method.

class BankServer {
  // Private constructor - cannot be called directly
  #constructor() {
    this.transactions = [];
    this.instanceId = BankServer.#generateId();
  }
  
  // Static method to get the singleton instance
  static getInstance() {
    if (!BankServer.#instance) {
      BankServer.#instance = new BankServer();
    }
    return BankServer.#instance;
  }
  
  // Other methods...
}
